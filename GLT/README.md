# GLT - Gelatina Que Seca

Pagina de vendas (landing page) do produto Gelatina Que Seca.

## Publicar no GitHub Pages
1. Suba estes arquivos neste repositorio (index.html + vsl.mp4).
2. Settings > Pages > Source: Deploy from a branch > main > /(root) > Save.
3. A pagina fica no ar em: https://SEU-USUARIO.github.io/GLT/

Antes de mandar trafego: trocar COLE_AQUI_O_LINK_DO_CHECKOUT (no index.html) pela URL real do checkout.
